Maui Jewelers

Project Overview

This is a website for a jewelry store, Maui Jewelers, in Las Vegas to sell its products online. The users will be customers of Maui Jewelers. There will be different pages for customers to choose from different jewelry items like watches to bracelets, and then there will also be a men and women section as well.

Dynamic Functionality Description

Interactive Product Showcase

Description: Implement interactive product showcases on the Home Page and individual product pages. This functionality will allow users to hover over product images to see alternate views or zoom in for a closer look. Additionally, clicking on a product image will open a modal window with more detailed product information and the option to add the item to the cart without leaving the page.

Example URLs:
Amazon: Amazon's product pages often feature interactive product images with zoom-in functionality.
Apple: Apple's website showcases its products with interactive images allowing users to view different angles.

Dynamic Filtering and Sorting

Description: Implement dynamic filtering and sorting options on category pages (e.g., Men's Jewelry, Women's Jewelry). Users can refine their product search based on criteria such as price range, material, size, and style. The page will update in real-time as users apply filters or change sorting preferences, providing a tailored browsing experience.

Example URLs:
Etsy: Etsy's category pages often offer dynamic filtering options for refining search results by various attributes.
Zappos: Zappos allows users to sort and filter shoe listings based on size, color, brand, and price range.

Interactive Cart Summary

Description: Enhance the Cart page with interactive features to provide users with a dynamic summary of their selected items. Users can adjust quantities, remove items, or update their cart directly within the summary section without needing to navigate away or refresh the page.

Example URLs:
Walmart: Walmart's cart page allows users to modify quantities and remove items directly within the cart summary.
Target: Target's cart summary dynamically updates as users make changes to their selections.

Live Search Suggestions

Description: Implement live search suggestions as users type in the search bar. As users input text, the website will display real-time suggestions based on matching product names or categories, helping users find what they're looking for more efficiently.

Example URLs:
Google: Google's search bar provides live suggestions as users type, enhancing the search experience.
YouTube: YouTube's search bar offers live suggestions for video titles and channels as users type.
